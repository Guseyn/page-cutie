# page-cutie

[![NPM Version][npm-image]][npm-url]

[Page](https://github.com/Guseyn/page) plugin for using cutie library in browser.

For more information visit [cutie](https://github.com/Guseyn/cutie).

## install

`npm install @page-libs/cutie`

## build

`npm run build`
